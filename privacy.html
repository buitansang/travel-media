Privacy Policy

Last updated: 21 July 2026

1. Scope

Travel Poster is operated by a single person for their own TikTok account. It has no other users, and it collects no data about any TikTok user other than the account owner who authorises it.

2. What data is handled

  • TikTok access and refresh tokens — issued when the account owner authorises the app, used solely to upload drafts to that same account.
  • Basic profile information (user.info.basic) — the account's display name and avatar, used only to confirm the correct account is connected.
  • Photographs supplied by the operator — travel photos the operator chooses to publish.

3. Where data is stored

Tokens are stored encrypted on the operator's own machine, inside a self-hosted n8n instance. They are never transmitted to any third party and never leave that machine except in requests to TikTok's own API endpoints.

Photographs intended for publication are hosted publicly on this GitHub Pages site, because TikTok's Content Posting API retrieves photo content by URL. Only photographs the operator has explicitly chosen to publish are placed here.

4. What is not done

No data is sold, shared, rented, or used for advertising. No analytics or tracking is performed on visitors to this site. No data about other TikTok users is requested, received, or stored.

5. Retention and deletion

Tokens are kept only while the integration is in use, and are destroyed when the operator revokes the app's access from their TikTok account settings. Published photographs can be removed from this site at any time by deleting them from the repository.
